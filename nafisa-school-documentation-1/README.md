## Project Description

The Nafisa Aldoo School Management System is designed to streamline the management of school operations, including student enrollment, class management, and staff administration. The system aims to enhance the educational experience by providing a user-friendly interface for teachers, students, and administrative staff.

## Installation

To set up the Nafisa Aldoo School Management System, please refer to the [Setup Guide](docs/deployment/setup-guide.md) for detailed instructions on prerequisites and installation steps.
